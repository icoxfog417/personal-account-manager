    領収書発行

    AWS では領収書を発行することはできかねるため、PDF 版の請求書をご利用ください。

    請求設定ページにて事前に「PDF 請求書を E メールで配信」を有効化いただくことで、PDF 版の請求書を E メールで受け取ることも可能です。また、支払いページからも PDF 版の請求書をダウンロードすることが可能です。

    解約後も請求書が届くのはなぜ？

    以下をご確認ください。

    https://repost.aws/ja/knowledge-center/closed-account-bill

    それでも解決しない場合は以下からお問合せください。

    https://pages.awscloud.com/contactus-account-support-jp.html

    AWSアカウントを譲渡したい

    以下のアカウント譲渡要件を満たしている場合につきましては、お客様側で譲渡処理を実行いただくことが可能となっております。

    AWSアカウント譲渡要件

    https://d1.awsstatic.com/legal/aws-account-assignment-requirements/AWS%20Account%20Assignment%20Requirements_JP_%202024-01-19.pdf

    自分の AWS アカウントを別の個人または企業に譲渡するにはどうすればよいですか?

    https://repost.aws/ja/knowledge-center/transfer-aws-account

    上記要件と合致しない場合、ご連絡いただきました情報にて同意書の作成を当社にて行い、ご締結が必要となります。

    この場合、同意書の発行まで2~3週間程のお時間をいただくこととなりますこと、ご了承いただけますと幸いです。

    要件に合致しない場合は営業担当までご連絡ください。

    ご利用中環境における不具合や障害のお問合せ

    ▼アカウントへログインできる（ケース起票できる）場合

    恐れ入ります、あいにくこちらの窓口では、いただきましたお問合せに関してご案内できかねてしまいます。

    お力になれず大変恐縮でございますが、下記弊社技術サポートまでお問い合わせくださいませ。

    ①技術サポートをご契約いただいている場合、以下手順にて技術サポートにお問合せくださいませ。

    • 当該AWSアカウントのコンソールにログイン( https://aws.amazon.com/jp/console/ )
    • 画面右上の「？」を選択
    • 「サポートセンター」を選択
    • 「ケースの作成（又は「Create Case」）」を選択
    • 「内容」で「技術サポート」を選択してお問合せを作成

    ②技術サポートをご契約いただいていない場合、有償のサポートプランにご加入いただく必要がございますので下記をご参照下さいませ。

    ご参考

    AWS サポートのプラン比較

    https://aws.amazon.com/jp/premiumsupport/compare-plans/

    AWS サポートのプランの料金

    https://aws.amazon.com/jp/premiumsupport/pricing/

    AWS ビジネスサポート

    https://aws.amazon.com/jp/premiumsupport/business-support/
